# Recipe API
This API interacts with recipes and recipe reviews from a MongoDB database

###What can it do?
<ul>
  <li>Access all recipes in the database</li>
  <li>Find recipe by ID</li>
  <li>Create reviews for recipes</li>
</ul>

###Built with:
<ul>
  <li>Java</li>
  <li>Spring Boot</li>
  <li>MongoDB</li>
</ul>
